# 00-06. Part 0 체크리스트

여기까지 다 체크되면 환경 세팅 끝. Git과 GitHub을 자유롭게 부릴 수 있는 상태입니다.

---

## ✅ 최종 체크리스트

### 도구 설치

- [ ] `git --version` 이 버전 숫자 출력 (macOS 또는 Windows Git Bash)
- [ ] VSCode가 깔려 있고 Source Control 패널을 열 수 있음

### Git 사용자 정보

- [ ] `git config --global user.name` → 내 이름 출력
- [ ] `git config --global user.email` → GitHub 가입 이메일과 **동일한** 이메일 출력
- [ ] `git config --global init.defaultBranch` → `main` 출력

### GitHub 계정

- [ ] github.com 로그인 가능
- [ ] 이메일 인증 완료
- [ ] **2FA 활성화**
- [ ] **백업 코드 16개를 안전한 곳에 저장** (1Password, 자기 메일 등)
- [ ] 프로필 사진·이름 설정 (선택이지만 권장)

### 인증

- [ ] 다음 명령이 SHA 목록을 출력 (인증 동작 확인):
      ```bash
      $ git ls-remote https://github.com/octocat/Hello-World.git
      ```
- [ ] (PAT을 발급한 경우) 만료일을 캘린더에 기록
