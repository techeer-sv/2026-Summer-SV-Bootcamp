# 8. Database — 핸즈온 실습

## 초급: SQL 기초 실습 - 도서 대여 데이터베이스

**목표**: 테이블 생성과 기본 SQL 명령(SELECT/INSERT/UPDATE/DELETE)을 직접 작성하며 익힌다.

**진행 순서**
1. SQLite 또는 MySQL Workbench 설치 및 연결
2. `books`, `members` 테이블을 `CREATE TABLE`로 생성
3. `INSERT`로 샘플 데이터 10건 이상 입력
4. 다양한 조건으로 `SELECT` 연습
   - "2020년 이후 출판된 책"
   - "제목에 특정 단어가 포함된 책" (`WHERE ... LIKE`)
   - "최신 출판일 순으로 5권만" (`ORDER BY`, `LIMIT`)
5. `UPDATE`/`DELETE`로 데이터 수정·삭제 연습

**결과물**
- 10개 이상의 쿼리와 실행 결과 캡처

## 고급: 전자상거래 데이터 분석 쿼리 작성

**목표**: JOIN, 집계, 트랜잭션, 인덱스/실행계획까지 실무에서 자주 쓰는 쿼리 패턴을 다룬다.

**진행 순서**
1. 샘플 데이터셋(`customers`, `orders`, `order_items`, `products`) import
2. JOIN으로 "고객별 총 구매금액" 조회 쿼리 작성
3. `GROUP BY` + `HAVING`으로 "5번 이상 주문한 고객" 찾기
4. 서브쿼리로 "평균 주문금액보다 높은 주문" 찾기
5. "주문 생성 + 재고 차감"을 하나의 트랜잭션으로 묶고, 중간에 실패시켜 `ROLLBACK` 동작 확인
6. `EXPLAIN`으로 느린 쿼리의 실행계획 확인 → 인덱스 추가 후 실행계획 재비교
7. (선택) 동일 데이터를 MongoDB 컬렉션 구조로 모델링하여 SQL 구조와 비교

**결과물**
- 분석 쿼리 5개 이상 + 결과
- 인덱스 적용 전/후 `EXPLAIN` 비교
